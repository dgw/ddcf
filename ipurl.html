DEFAULT IP URL

I advise against changing this unless you know what you are doing. It requires you to search into the code to find certain things and change them. If you still wish to continue, read on. Otherwise, leave it as None.

We use Python's httplib for handling the IP request.

Changing the option


Do not include any `/' or the `http://' portions of the URL. Domain only. 
defaultIPURL = "example.com"

Changing the request


Finding the change places is relatively easy. Search for `</script>/n'. The change points are generally around this. Making the change isn't as easy.

Changing the location
The URL of the script with the IP. This is the same syntax as any HTTP request, minus the protocol identifier. 
ipck.request("GET","/")
The second parameter is generally good enough to change.

Changing to HTTPS
Simply change the lines... 
ipck = httplib.HTTPConnection(ipurl)
to... 
ipck = httplib.HTTPSConnection(ipurl)

Parsing the output


Returning plain text
If you're just getting plain text with no frills, you can get away with using 
ip = ip2

Returning valid HTML
If you're returning valid HTML (meaning within a body tag), you can use this... 
ip = ip2.split("<body>")[1].split("</body>")[0]
You might have to adjust this for newlines, as this is fed directly into the API call.
