WHAT IS DDCF?

ddcf is a cross-platform command line Dynamic DNS update client for CloudFlare. Dynamic DNS is used when your server's IP isn't the same all the time. (or not static, it's a dynamic IP.) It's extremely easy to use, and doesn't require you to do much of anything beyond executing the program. It's designed to be extremely fast, and uses the fastest modules I could find for it to use.

FEATURES

  • Updates DNS zones
  • Adds A and CNAME DNS zones
  • Removes DNS zones
  • Shows account stats

REQUIREMENTS

  • Python v2 with SSL support
  • Internet connection
  • CloudFlare account and API key

SETUP

Setup is extremely simple and quick.
Open up the script in your favorite text editor (Notepad, Gedit, Kate, etc. Do not use WordPad, however.) Go to line 16, and change the Email and API key to your credentials from CloudFlare. You may want to set a default hostname, also. The 3rd and 4th options do not have to be set.
Example: 
# SETUP
#--------------------
# These HAVE to be set.
emailAddress		= "you@example.com"	# Set this to your CloudFlare Email.
APIkey 			= "Your API Key"	# Set this to your API key.
# These do not have to be set.
defaultHostname		= "example.com"		# Set this to your default hostname
defaultIPURL		= None			# This doesn't need to be changed. (Seriously.) It requires you to recode this script.
Not kidding on the last setting, don't change it.
